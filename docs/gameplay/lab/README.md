# Gameplay Lab

Laboratoire d'idées gameplay. Ces documents ne sont pas des specs MVP et ne doivent pas être traités comme des décisions implémentables tant qu'ils ne sont pas promus dans les docs gameplay numérotées.

Objectif : explorer les pistes qui pourraient moderniser Battle for the Crown sans trahir le coeur Travian / Kingsage / Tribal Wars.

## Pistes ouvertes

| Sujet | Statut | Intention |
| --- | --- | --- |
| [Rétention mobile moderne](./mobile-retention-modernization.md) | À creuser en premier | Transformer quêtes, Oyez et pass en système quotidien lisible, scalable, non-snowballant. |
| [Tribus](./tribes.md) | Idée courte | Poser l'importance sociale long terme sans spécifier encore le système complet. |
| [Barbares comme pilier PVM](./barbarians-pvm-pillar.md) | Idée courte | Faire des barbares un vrai pilier solo, pas seulement une réserve de loot. |
| [Mondes à règles spéciales](./special-world-rules.md) | Idée courte | Varier les saisons avec des rulesets spéciaux pour renouveler le métagame. |
| [Tickets d'exploration](./tickets/) | Backlog court | Garder les idées utiles à approfondir sans les promouvoir en specs. |

## Règle de promotion

Une idée du lab peut devenir une spec seulement si elle a :

- un objectif joueur clair ;
- un impact économique identifié ;
- des garde-fous anti-snowball ;
- un lien explicite avec les docs canoniques concernées ;
- une décision de scope : MVP, Saison 1, post-MVP, ou abandon.

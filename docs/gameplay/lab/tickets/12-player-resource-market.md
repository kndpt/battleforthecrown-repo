# 12 — Marché entre joueurs

**Statut** : idée à haut risque, à garder sur table  
**Ratio bénéfice / coût** : incertain

## Opportunité

Un bâtiment de marché pourrait permettre d'échanger bois / pierre / fer entre joueurs. Cela donnerait un rôle économique aux joueurs, renforcerait les relations sociales, et réduirait certains blocages de progression.

## Risque principal

Le marché est une porte ouverte aux abus :

- feed d'un gros compte par des petits comptes ;
- multi-comptes / alts ;
- transfert de ressources après menace ou racket ;
- contournement du pillage et de la distance ;
- accélération de Seigneur / conquête ;
- économie difficile à équilibrer ;
- support/modération plus lourds.

## Piste prudente

Si le sujet revient, explorer d'abord une version très limitée :

- échange uniquement via bâtiment Marché ;
- taux défavorable ou taxe forte ;
- limites journalières ;
- pas de transfert de couronnes ;
- distance maximale ;
- délai de transport attaquable ou interceptable seulement plus tard ;
- trading peut-être limité aux amis/tribu, mais cela renforce aussi le risque de feed.

## Alternative plus saine

Avant le marché joueur-joueur, tester un **échange royal limité** :

- conversion bois / pierre / fer contre taux défavorable ;
- plafond quotidien ;
- pas d'interaction sociale ;
- réduit les blocages sans créer d'économie exploitable.

## Points à trancher

- Le marché est-il vraiment nécessaire si pillage, sites d'exploitation et quêtes existent ?
- Doit-il être post-MVP strict ?
- Peut-il exister sans transfert direct joueur-joueur ?
- Quels signaux anti-abus seraient nécessaires ?

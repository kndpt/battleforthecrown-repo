# Barbares comme pilier PVM

> Laboratoire d'idée. Sujet volontairement court : il pose l'ambition, pas encore la spec.

## Idée

Les barbares ne doivent pas être seulement des sacs de loot. Ils doivent devenir un vrai pilier PVM capable de faire tenir un joueur solo plusieurs jours, même avant que le PvP devienne central.

Le joueur doit pouvoir se dire : "je peux jouer ce soir sans attaquer un humain, et j'ai quand même une vraie session de stratégie".

## Pourquoi

Le PvP est le coeur long terme, mais il est fragile en early game :

- tous les joueurs ne veulent pas attaquer humainement dès le premier jour ;
- le monde peut être peu dense au lancement ;
- un joueur solo doit avoir une boucle intéressante avant la tribu ;
- le jeu doit rester vivant même sans conflit humain immédiat.

Les barbares peuvent remplir ce rôle si leur écosystème devient lisible, progressif et stratégique.

## Direction à explorer

Pistes possibles :

- camps barbares avec identités différentes ;
- chefs barbares régionaux ;
- routes de pillage PVM ;
- villages barbares qui se régénèrent différemment selon leur tier ;
- Oyez centrés sur les barbares ;
- raids barbares défensifs plus narratifs ;
- objectifs PVM de plusieurs jours ;
- conquête barbare qui ouvre une vraie décision de placement, pas juste un village bonus.

## Garde-fou

Le PVM ne doit pas remplacer le PvP. Il doit préparer au PvP :

- apprendre les compositions ;
- valoriser le scout ;
- enseigner les temps de trajet ;
- habituer aux pertes ;
- créer des points de friction sur la carte ;
- attirer plusieurs joueurs vers les mêmes zones.

## Refactor

### Shared

- `packages/shared/src/auth/display-name.ts` — nouveau helper pur `formatAnonymousPlayerName(userId)` (gère `null | undefined | ''` → `Joueur ?`).
- `packages/shared/src/auth/display-name.spec.ts` — 4 cas (`-6` slice, court, null, vide).

### Backend

- **Nouveau module pur** `battleforthecrown-backend/src/common/display-names.ts` :
  - `loadUserDisplayNames(reader, userIds)` — `PrismaClientOrTx` compatible, skip DB si input vide.
  - `resolvePublicPlayerName(userId, displayNames)` — fallback shared.
  - `resolveWorldDisplayName(reader, worldId)` — parse identity → fallback `world.name` → fallback `worldId`.
- `display-names.spec.ts` — 3 describes (loadUserDisplayNames × 3, resolvePublicPlayerName × 3, resolveWorldDisplayName × 3).
